# 操作符

在前面的章节中，我们已经对操作符有了一定的了解，现在，我们来对它进行详细的介绍。

## 内置操作符

我们已经知道，Ktorm 的内置操作符按照实现方式大概可以分为三类：

- 使用 operator 关键字重载的 Kotlin 内置操作符
- 使用 infix 关键字创建的自定义操作符，如 `and`、`or`、`like` 等
- 使用普通 Kotlin 函数实现的一元操作符，如 `isNull`
